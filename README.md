# ⚖ Legal Document Simplifier AI

An AI-powered Streamlit application that helps users understand complex legal documents using Gemini AI.

---

## 🚀 Features

- Upload PDF, DOCX, TXT files
- AI-powered document summarization
- Legal language simplification
- Clause-by-clause explanation
- RAG-based AI chatbot
- Multi-language translation
- Activity history tracking

---

## 🧠 Tech Stack

- Python
- Streamlit
- Google Gemini AI
- FAISS (Vector Database)
- Sentence Transformers
- LangChain
- PyMuPDF / pdfplumber

---

```bash
git clone <your-repo>
